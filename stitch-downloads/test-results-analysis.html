Question 2 close Incorrect

Which sorting algorithm has the worst-case time complexity of O(n^2)?

Your Answer

cancel Merge Sort

Correct Answer

check_circle Quick Sort

Explanation

While Quick Sort is typically very fast (O(n log n) on average), its worst-case time complexity is O(n^2) when the pivot chosen is consistently the smallest or largest element (e.g., when the array is already sorted and a poor pivot strategy is used). Merge Sort, on the other hand, guarantees O(n log n) in all cases.
